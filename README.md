# K12 销售工具集

两个面向教培销售的 Streamlit 应用：

## 1. K12 教培客户画像分析器（`app.py`）

输入家长/孩子信息，输出客户画像、购买意向评分、跟进策略和话术建议。
支持 Gemini AI 分析（配 API Key）和离线规则分析双模式。

```bash
streamlit run app.py
```

## 2. 高中网课销售追踪助手（`sales_tracker.py`）

面向高中（高一/高二/高三）在线课程销售的客户追踪 CRM：

- **📋 今日工作台**：今日/逾期/未来3天跟进任务，7天未动客户流失预警
- **👥 客户管理**：客户档案录入与维护，自动计算意向评分和画像类型，支持 Excel 导出
- **📝 客户跟进**：记录每次沟通（方式/内容/结果），自动安排下次跟进时间
- **💡 推销助手**：按客户画像生成网课专属话术、阶段行动建议、异议应对、智能跟进计划（可一键写入跟进时间）
- **📈 数据看板**：销售漏斗、转化率、来源渠道分析、高意向客户排行

```bash
streamlit run sales_tracker.py
```

客户数据保存在本地 `sales_tracker.db`（SQLite），不上传。

## 安装依赖

```bash
pip install -r requirements.txt
```
